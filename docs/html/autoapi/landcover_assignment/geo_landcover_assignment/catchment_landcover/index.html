landcover_assignment.geo_landcover_assignment.catchment_landcover

This module provides a class for managing and analyzing land cover data within catchment areas. It integrates various APIs and data sources to offer a comprehensive set of functionalities for land cover analysis.

Classes

class landcover_assignment.geo_landcover_assignment.catchment_landcover.CatchmentLandCover[source]

This class is designed to access and analyze land cover data across different land use types within catchment areas. It provides methods to calculate areas and shares of different land cover types, including forests, wetlands, croplands, and grasslands, based on catchment names.

get_area_with_organic_potential(spared_breakdown, total_spared_area, sc)[source]

Calculates the area with organic farming potential based on spared land breakdown and scenario.

This method assesses the potential for organic soils in spared areas by analyzing the soil composition and other factors. It uses detailed breakdown data of spared areas and soil groups. The input data is calculated in the grassland_production module. It is assumed that the area of organic soils cannot be greater than the area of available soil group 3.

Parameters:

  • spared_breakdown (pd.DataFrame) – A pandas DataFrame containing detailed breakdown of spared areas, including soil types.

  • total_spared_area (pd.DataFrame or pd.Series) – A pandas DataFrame or Series containing total spared areas for various scenarios.

  • sc (str or int) – The scenario identifier used for the analysis.

Returns:

The area with available organic soils based on the specified scenario.

Return type:

float

Raises:

ValueError – If the specific scenario does not exist in the spared_breakdown or total_spared_area data.
