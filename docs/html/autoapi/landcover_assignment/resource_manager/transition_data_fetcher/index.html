landcover_assignment.resource_manager.transition_data_fetcher

The TransitionDataFetcher class is designed to process and analyze transition data from grasslands to various land uses based on specific scenarios. This class simplifies the process of extracting transition area data and supports detailed environmental and land use planning analyses.

class landcover_assignment.resource_manager.transition_data_fetcher.TransitionDataFetcher(transition_data)[source]

Initializes the TransitionDataFetcher with a pandas DataFrame containing transition data between different land use types.

Parameters:

transition_data – A pandas DataFrame containing transition matrices for various scenarios.
